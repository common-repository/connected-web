Connected Web is a web syndication format focused on high quality and cross device content sharing.

== Description ==

Connected Web is an open source web syndication format to share data for aggregation. Any web source can use it to publish contents and help third party platforms to aggregate and distribute their data in the most efficient way. A content can be any kind of static data produced by a source, such as full articles, videos, pictures, galleries, weather forecasts data and more.
